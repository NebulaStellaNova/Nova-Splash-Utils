## WARNING!!

For this mod to work you need to install genzu's Extra Utils mod, which you can find here: https://github.com/g3n-zu/extra-utils

## How To Use

To use this mod create a file in `YourMod/data/` called `splashscreen.json` and reference this mod's `splashscreen.json`.
